# Техническая спецификация AI-торговой системы для BingX

## Обзор проекта
Создание искусственного интеллекта для автоматической торговли фьючерсами на платформе BingX с использованием API-интеграции. Система будет поддерживать различные режимы торговли (скальпинг, спот, долгосрочные сделки) с настраиваемым управлением рисками.

## Архитектура системы

### 1. API-модуль (BingX)
- Интеграция с BingX API для торговли фьючерсами
- Аутентификация через API-ключи
- Получение рыночных данных
- Выполнение ордеров

### 2. Модуль технического анализа
- RSI (Relative Strength Index)
- MACD (Moving Average Convergence Divergence)
- Полосы Боллинджера
- Стохастический осциллятор
- SMA/EMA (Simple/Exponential Moving Average)
- Уровни поддержки/сопротивления

### 3. Модуль управления торговлей
- Режим скальпинга (1-5 минут)
- Режим спот (15-30 минут)
- Режим долгосрочных сделок (1-4 часа)
- Управление плечом
- Контроль рисков и размера позиции

### 4. Модуль ИИ/машинного обучения
- Алгоритмы классификации для определения сигналов
- Обучение на исторических данных
- Оптимизация параметров торговли
- Ранжирование активов по потенциалу

### 5. Модуль управления рисками
- Настройка максимального риска на сделку
- Управление капиталом (например, использовать только 29% от $100)
- Стоп-лосс и тейк-профит
- Мониторинг общего P&L

### 6. Графический интерфейс
- Отображение текущих сигналов
- Управление режимами торговли
- Настройка параметров риска
- Мониторинг статуса системы

## Функциональные возможности

### Торговые режимы
1. **Скальпинг**
   - Временные фреймы: 1-5 минут
   - Частые сделки с низким риском
   - Быстрое принятие решений

2. **Спот**
   - Временные фреймы: 15-30 минут
   - Среднесрочные сделки
   - Баланс между риском и прибылью

3. **Долгосрочные сделки**
   - Временные фреймы: 1-4 часа
   - Меньше сделок, больше прибыли на сделку
   - Анализ на основе нескольких таймфреймов

### Управление капиталом
- Возможность указания максимального процента капитала для использования (например, 29% от $100)
- Автоматический расчет размера позиции
- Контроль общего риска портфеля

### Сигналы и точки входа/выхода
- Автоматическое определение точек входа на основе ИИ
- Рекомендации по тейк-профиту и стоп-лоссу
- Возможность автоматического исполнения или ручного подтверждения
