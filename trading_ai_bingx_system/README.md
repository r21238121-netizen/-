# AI Торговая Система для BingX

## Описание

AI-торговая система для автоматической торговли фьючерсами на платформе BingX. Система использует технический анализ и управление рисками для принятия торговых решений.

## Особенности

- Интеграция с API BingX
- Три режима торговли: скальпинг, спот, долгосрочные сделки
- Технический анализ с использованием RSI, MACD, полос Боллинджера и других индикаторов
- Управление рисками с ограничением капитала (например, использовать только 29% от $100)
- Графический интерфейс для мониторинга и управления
- Автоматическое или рекомендательное исполнение ордеров

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Настройте API-ключи в `config/config.py`

## Настройка API-ключей

1. Зайдите на сайт BingX и создайте API-ключи с разрешением на торговлю фьючерсами
2. Откройте файл `config/config.py`
3. Замените пустые строки для `BINGX_API_KEY` и `BINGX_SECRET_KEY` на ваши ключи

## Запуск

### GUI режим (рекомендуется):
```
python run_trading_ai.py
```

### CLI режим:
```
python run_trading_ai.py --cli --mode scalping
```

### С указанием режима:
```
python run_trading_ai.py --mode spot  # или long_term
```

### С автоматическим исполнением ордеров:
```
python run_trading_ai.py --auto-exec
```

## Режимы торговли

1. **Скальпинг** (`scalping`): 
   - Таймфрейм: 1 минута
   - Интервал проверки: 30 секунд
   - Плечо: 10x
   - Риск на сделку: 1%

2. **Спот** (`spot`):
   - Таймфрейм: 15 минут
   - Интервал проверки: 5 минут
   - Плечо: 5x
   - Риск на сделку: 2%

3. **Долгосрочные сделки** (`long_term`):
   - Таймфрейм: 1 час
   - Интервал проверки: 30 минут
   - Плечо: 3x
   - Риск на сделку: 3%

## Управление капиталом

Система по умолчанию настроена на использование 29% от $100 ($29) для торговли. Эти параметры можно изменить в `config/config.py`:

- `TOTAL_BALANCE`: общий баланс
- `RISK_CAPITAL_PERCENT`: процент капитала для риска
- `MAX_POSITION_SIZE_PERCENT`: максимальный размер позиции (10% от рискового капитала)

## Безопасность

- Система включает встроенные ограничения риска
- Рекомендуется начинать с небольших сумм
- Используйте только средства, которые можете позволить себе потерять
